README.txt

1) Start a local server in this current directory using python 3
example command in terminal: python -m http.server 8000

2) Open your browser and enter the address: http://localhost:8000

3) You will now see the visualization for life expectancy vs income in graphical view

4) Use the time slider at the bottom by clicking and dragging the slider button to the preffered year and observe the change in data

5) Hover over countries to view their label and observe the trajectory over time

6) Refer to the legend on the right hand side to identify countries at a regional level

7) Use the green buttons to navigate between different datasets

8) Use the red button to switch to world view

	8.1) The green button in world view will return to life expectancy graph view
